# Mountain Store 🏔️

A full stack e-commerce web application for outdoor adventure products.

## Developer
**Ijlal Ahmed** - 2500008

## Tech Stack
- **MongoDB Atlas** - Database
- **Express.js** - Backend Framework
- **Angular 4** - Frontend Framework
- **Node.js** - Runtime Environment

## Features
- 6 Outdoor Categories (Hiking, Camping, Climbing etc.)
- 7 Products with prices
- User Login & Registration (JWT Authentication)
- Shopping Cart
- Beautiful outdoor mountain theme
- Responsive design with Bootstrap

## How to Run

### Backend (Terminal 1)
nvm use 20.19.0
cd server
node server.js

### Frontend (Terminal 2)
nvm use 8.17.0
cd client/Angularecommerce
node_modules\.bin\ng serve

Open browser: http://localhost:4200
